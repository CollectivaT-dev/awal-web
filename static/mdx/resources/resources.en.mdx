<ul className="list-disc ml-4">

<li>Datasets collected as part of Awal project can be found in Col·lectivaT's [Hugging Face de Col·lectivaT](https://huggingface.co/datasets/collectivat/amazic)  </li>
<li>NLLB-200 is a set of multilingual machine translation models created by [Meta](https://huggingface.co/facebook/nllb-200-distilled-600M). It's the first open source model that includes Tamazight. The version used in this website is the 600 million parameter model. </li>
<li>Tamazight NLP group publishes open datasets and models in [Hugging Face](https://huggingface.co/Tamazight-NLP)</li>
<li>[Tamazight wikipedia](https://zgh.wikipedia.org/wiki/%E2%B5%9C%E2%B4%B0%E2%B5%99%E2%B5%8F%E2%B4%B0_%E2%B5%8F_%E2%B5%93%E2%B5%99%E2%B5%8F%E2%B5%93%E2%B4%B1%E2%B4%B3)</li>
<li>[TALAM portal](https://tal.ircam.ma/talam/) groups applications in relation with automatic processing of Tamazight.</li>
<li>[Darija Open Dataset](https://darija-open-dataset.github.io/) is an open source collaborative project for Darija-English machine translation. Awal's Darija sentence database is based on this dataset. </li>
<li>[Tamazight For All](https://www.youtube.com/@TamazightForAll?si=v465LDveFfj6U0Fu) is a YouTube channel where you can find lessons, discussions, and short stories in Tamazight, along with insights into Amazigh culture.</li>
</ul>
