<ul className="list-disc ml-4">

<li>El proyecto Awal se puede encontrar en el repositorio de [Hugging Face de Col·lectivaT](https://huggingface.co/datasets/collectivat/amazic)  </li>
<li>NLLB-200 es un conjunto de modelos de traducción automática multilingües creados por [Meta](https://huggingface.co/facebook/nllb-200-distilled-600M). Es el primer modelo de código abierto que incluye el amazigh. La versión utilizada en este sitio web es el modelo de 600 millones de parámetros.</li>
<li>El grupo Tamazight NLP publica conjuntos de datos y modelos abiertos en [Hugging Face](https://huggingface.co/Tamazight-NLP)</li>
<li>[Wikipedia en Tamazight](https://zgh.wikipedia.org/wiki/%E2%B5%9C%E2%B4%B0%E2%B5%99%E2%B5%8F%E2%B4%B0_%E2%B5%8F_%E2%B5%93%E2%B5%99%E2%B5%8F%E2%B5%93%E2%B4%B1%E2%B4%B3)</li>
<li>[El portal TALAM](https://tal.ircam.ma/talam/) recopila aplicaciones relacionadas con el procesamiento automático del Tamazight </li>
<li>[Darija Open Dataset](https://darija-open-dataset.github.io/) es un proyecto colaborativo de código abierto para la traducción automática darija-inglés. La base de datos de oraciones en darija de Awal se basa en este conjunto de datos.</li>
<li>[Tamazight For All](https://www.youtube.com/@TamazightForAll?si=v465LDveFfj6U0Fu) es un canal de YouTube donde puedes encontrar lecciones, discusiones y cuentos cortos en tamazight, junto con reflexiones sobre la cultura amazigh.</li>
</ul>
