title: 'Recursos'

<ul className="list-disc ml-4">

<li>Le projet Awal peut être trouvé dans le référentiel [Hugging Face de Col·lectivaT](https://huggingface.co/datasets/collectivat/amazic)  </li>
<li>NLLB-200 est un ensemble de modèles de traduction automatique multilingues créés par [Meta](https://huggingface.co/facebook/nllb-200-distilled-600M).C'est le premier modèle open source qui inclut le tamazight. La version utilisée sur ce site web est le modèle de 600 millions de paramètres.</li>
<li>Le groupe Tamazight NLP publie des ensembles de données et des modèles ouverts sur [Hugging Face](https://huggingface.co/Tamazight-NLP)</li>
<li>[Wikipédia en Tamazight](https://zgh.wikipedia.org/wiki/%E2%B5%9C%E2%B4%B0%E2%B5%99%E2%B5%8F%E2%B4%B0_%E2%B5%8F_%E2%B5%93%E2%B5%99%E2%B5%8F%E2%B5%93%E2%B4%B1%E2%B4%B3)</li>
<li>[Le portail TALAM ](https://tal.ircam.ma/talam/) regroupe des applications liées au traitement automatique du Tamazight </li>
<li>[Darija Open Dataset](https://darija-open-dataset.github.io/)  est un projet collaboratif open source pour la traduction automatique darija-anglais. La base de données de phrases en darija d'Awal est basée sur cet ensemble de données. </li>
<li>[Tamazight For All](https://www.youtube.com/@TamazightForAll?si=v465LDveFfj6U0Fu) est une chaîne YouTube où vous pouvez trouver des leçons, des discussions et des histoires courtes en Tamazight, ainsi que des réflexions sur la culture amazighe.</li>
</ul>
